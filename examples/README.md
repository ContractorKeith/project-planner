# Example Outputs

These files show what a completed planning session produces. The fictional project
is **TaskTrail** — a simple CLI task tracker built with Python.

These are _examples only_. They are not used by the orchestrator. Browse them to
see what your own `/outputs/` folder will look like after a session.

| File | What it is |
|---|---|
| `PRD.md` | Product requirements document |
| `STACK.md` | Technology decisions with rationale |
| `TASKS.md` | First sprint task list |
| `CLAUDE.md` | Context file for the build repo |
